# 📊 E-Commerce Sales Analysis using SQL

## 📌 Project Overview

This project analyzes the Global Superstore dataset using MySQL to solve real-world business problems. It focuses on sales performance, customer behavior, product profitability, and regional analysis through SQL queries.

A total of **50 business questions** were solved using SQL to demonstrate practical data analysis skills.

---

## 📂 Dataset Information

- **Dataset:** Global Superstore
- **Total Records:** 24,721
- **Unique Customers:** 1,586
- **Categories:** 3

## 💡 Key Insights

- Technology generated the highest sales revenue.
- Technology was the most profitable category.
- The United States contributed the highest overall sales.
- Phones and Copiers were among the best-performing product groups.
- Sales showed steady year-over-year growth.
- Customer and product ranking helped identify the highest-value contributors.
